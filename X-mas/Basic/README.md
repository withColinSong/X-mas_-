# 1. HTML

## 1.1 HTML 기본 뼈대
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
## 1.2. HTML 태그
- `head` - 우리가 보이지 않는 영역
- `body` - 우리가 보이는 영역
- `div` - 영역을 나눈다.
- `p` - 문단 태그
- `h1~h6` - 제목 태그
- `span` - 특정 글자
- `a` - 링크 태그
- `textarea` - 긴 글을 작성할 수 있는 태그

# 2. CSS

- `class` - 태그도 이름표를 붙여주는 작업이 필요하다.
- `padding` - 내 안쪽의 여백 
- `margin` - 내 바깥의 여백
